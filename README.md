# Codex Arcade

Codex Arcade is a multi-game arcade collection.

## Games

- `flamethrow/`: neon Three.js basketball arcade game.
- `MortalCodex/`: Phaser arcade fighting game.
- `drift/`: Three.js time-trial racing game built around short-hop drift slides.

## Run a game

```sh
cd flamethrow
npm install
npm run dev
```

```sh
cd MortalCodex
npm install
npm run dev
```

```sh
cd drift
npm install
npm run dev
```
